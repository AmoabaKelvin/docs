## What is ishortn?

iShortn is a powerful URL shortening and QRCode service designed to make sharing links
easier and more efficient. Whether you're managing marketing campaigns,
tracking click statistics, or simply sharing links, iShortn offers a robust
solution with premium features for advanced users.

## Features

- Link shortening
- Link analytics (clicks, countries, devices, browsers, cities)
- Custom domain support
- Link expiration
- Link QR code generation
- Link password protection
- Geo targeting (coming soon)
- And many more
